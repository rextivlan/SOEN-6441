# YouTrack

Use this application to track various YouTube videos!

## Install Dependencies

### Frontend dependencies

cd frontend\
npm install

### Backend dependencies

cd backend\
npm install

## Run

### Run frontend

cd frontend\
npm start

### Run backend(server)

cd backend\
npm start

### Run tests

cd backend\
npm run tests
